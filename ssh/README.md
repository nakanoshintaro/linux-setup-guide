# SSH 設定と活用ガイド（ssh/README.md）

このディレクトリでは、Ubuntu 環境における SSH（Secure Shell）の導入、設定、接続、トラブル対処法をまとめています。

---

## 🛠 主な内容

- SSH サーバのインストールと有効化方法
- パスワード認証・鍵認証の設定
- クライアントからの接続方法（VSCode含む）
- セキュリティ向上のための推奨設定
- トラブルシューティング（接続できない、タイムアウトなど）
